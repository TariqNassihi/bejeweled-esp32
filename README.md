# Bejeweled for ESP32

A Bejeweled-style puzzle game implemented in C for the ESP32 microcontroller.  
The game runs on an ESP32 with an integrated display and uses hardware buttons for user input.

## Description

This project implements a match-3 puzzle game similar to Bejeweled.  
Players swap adjacent gems to form matches of three or more identical elements, which are then cleared from the board.

The game is designed to run on an ESP32 microcontroller and interacts directly with the hardware display and input buttons.
